**AI-Driven HR Operations & Employee Engagement System**

This project is an AI-powered HR automation system built using Salesforce and Agentforce during a 24-hour hackathon.
The goal is to simplify HR workflows and improve employee engagement using intelligent automation and AI agents.

**Problem**
- HR teams often spend significant time on repetitive tasks such as:
- Collecting and analyzing employee feedback
- Monitoring employee engagement
- Identifying burnout risks
- Managing employee referrals
- Organizing internal events
These manual processes slow down decision making.

**Solution**
This system introduces AI-based agents and automation flows to analyze feedback, detect burnout risks, automate referrals, and assist HR in managing employee engagement activities.

**Key Features**
- Feedback sentiment analysis
- AI burnout detection
- Referral management system
- Birthday assistant automation
- Employee engagement support

**Tech Stack**
- Salesforce Flow
- Agentforce AI Agents
- Prompt Builder
- Custom Salesforce Objects
- Automation Workflows
